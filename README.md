# PRANASYS

Trabajo Práctico Integrador - Análisis de Sistemas

PRANASYS es una aplicación orientada a la organización y gestión de retiros holísticos, espirituales y de bienestar.

## Objetivo

Centralizar la información relacionada con retiros, facilitando la búsqueda, organización, seguimiento de pagos, gestión de acompañantes y valoración de experiencias.
